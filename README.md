## 为什么要改写这个项目?

MyBatis-Plus在最近的迭代版本中，本人已经没有怎么去迭代关注（由于工作较忙的原因），在最近的版本过程中，发现还是有一些不太完善的地方（核心功能还是比较nice的）。所以我在提出这些优化问题之后，很遗憾目前的MyBatis-Plus团队没有采纳我的观点，既然用的不爽所以我决定维护一个自己和自己公司的一个使用版本
